O Pure CSS foi criado pela equipe do Yahoo, essa feramente tem módulos que você pode utilizar em praticamente qualquer projeto CSS. Será totalmente possível criar botões, grids, tabelas, menus e vários outros recursos responsivos de maneira extremamente fácil.

Layout de menu lateral responsivo


Layout de menu lateral responsivo é um tipo de barra de menu responsiva no Pure.CSS, que altera a barra de menu horizontal para uma oculta botão de alternância quando o tamanho da tela é reduzido. Quando a largura do site muda de dispositivo para dispositivos, como em tablets e smartphones, é usado para fazer a barra de menu responsivo. O menu responsivo Pure.CSS não precisa de nenhuma função JavaScript, é classes CSS embutidas são suficientes para isso.
